# Persona

Web Components with CSS

## NPM

```bash
yarn add @elex-project/persona
```

## What I did

### 프로젝트 만들기

```bash
yarn init
yarn add webpack webpack-cli --dev
yarn webpack-cli init
```

### Lit

```bash
yarn add lit
```

### tsconfig.json

```json
"compilerOptions": {
    ...
    "target": "es6",
    "moduleResolution": "node",
    "experimentalDecorators": true
  }
```

### Storybook

```bash
yarn add sb --dev
yarn sb init
yarn storybook
```
